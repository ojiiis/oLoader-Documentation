Introduction

oLoader is a lightweight JavaScript framework designed for dynamically loading HTML, CSS, and JavaScript content into a webpage efficiently.

Key Features

  • Dynamic content loading
  • Script management
  • Head and body content injection
  • Efficient queuing system
  • Supports both inline and external files
